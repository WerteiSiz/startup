# Шаг 1. Гипотезы монетизации  

## 1.1 Три альтернативные модели  

| ID | Модель | Кто платит | За что | Единица оплаты | Цена |
|----|--------|-----------|--------|----------------|------|
| M1 (приоритет) | Комиссия с партнёров | Заведения | Приведённые клиенты | За заказ | 10–20% |
| M2 | B2B-подписка | Бизнес | Размещение и продвижение | За месяц | 990–4 990 ₽ |
| M3 | B2C-подписка | Пользователь | Расширенные возможности | За месяц | 199 ₽ |

---

## 1.2 Приоритетная модель  

**Выбор:** M1 — комиссия с партнёров  

**Обоснование:**
- Нет барьера для пользователя (бесплатно)
- Прямая монетизация от действий (привёл клиента → заработали)
- Масштабируемая модель
- Соответствует рынку агрегаторов скидок и сервисов бронирования

# Шаг 4. Финансовая модель  

## 4.1 Допущения  
- 1000 пользователей / месяц  
- 10% переходят  
- Средний чек: 1000 ₽  
- Комиссия: 15%  

---

## 4.2 Доход
