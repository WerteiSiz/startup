# 1.  Ядро роста и стратегия

## 1.1 Суть продукта

**Наш продукт:**  
Платформа скидок и предложений на досуг и развлечения:
- антикафе  
- квесты  
- бары  
- активности и ивенты  
- городские развлечения  

 Пользователь находит места со скидками, бронирует и посещает их через платформу.

## 1.2 Основной драйвер роста

###  Growth-модель: PLG + локальная виральность

Каждое посещение места = социальное событие:
Пользователь приходит через скидку
↓
Посещает место (досуг / ивент / квест)
↓
Делится опытом с друзьями
↓
Друзья переходят в сервис
↓
Они тоже используют скидки
↓
Цикл повторяется

 

##  Ключевая метрика роста

- k-factor (виральность):  
  - 3 месяца → ≥ 0.4  
  - 12 месяцев → ≥ 0.8

# 2.  Стратегия роста (эксперименты)
