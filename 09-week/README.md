# Шаг 1. Гипотезы теста и критерии успеха

## 1.1 Сегмент, сценарий, ценность

| Параметр | Значение |
|----------|---------|
| Сегмент | Пользователи 18–30 лет, активно использующие цифровые сервисы |
| Сценарий | Пользователь заходит на сайт, регистрируется и начинает использовать продукт |
| Ценностное предложение | Быстрое и удобное решение пользовательской задачи |
| Целевое действие | Успешное использование ключевого функционала продукта |

---

## 1.2 Список гипотез

| ID | Тип | Гипотеза | Критерий успеха | Критерий провала |
|----|-----|----------|----------------|-----------------|
| H1 | Ценность | Пользователи понимают и принимают ценность продукта | ≥ 70% положительных отзывов | < 50% |
| H2 | Сценарий | Пользователь проходит основной сценарий без помощи | ≥ 70% завершений | < 50% |
| H3 | UX | Интерфейс понятен и удобен | ≤ 30% жалоб | ≥ 50% жалоб |
| H4 | Retention | Пользователь готов вернуться | ≥ 50% повторного использования | < 30% |
| H5 | Монетизация | Пользователь готов платить | ≥ 25% готовы платить | < 15% |
