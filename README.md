# 📂 Описание стартап-идей

## 1️⃣ K9 Match — «Кинологический Tinder»

### Проблема
Кинологические службы (МВД, таможня, МЧС) тратят годы на подбор генетически совместимых пар собак.  
Родословные хранятся разрозненно и часто в бумажном виде.

### Решение
Создание закрытой цифровой платформы:

- blockchain-реестр родословных
- AI-алгоритм подбора пар
- прогнозирование рабочих качеств потомства

### Ценность
- ускорение селекции
- снижение затрат на подготовку собак
- аналитический подход к разведению

### Бизнес-модель
- подписка питомников
- лицензии государственным структурам
- аналитические отчёты

### Стоимость MVP
≈ 700 000 ₽

## 2️⃣ MedIQ / ФармаПрофиль

### Проблема

#### Разрыв «Сканер vs Медицина»
Приложения проверки лекарств показывают инструкцию, но не учитывают состояние здоровья пользователя.

#### Полипрагмазия
Пользователи принимают несколько препаратов одновременно:
- сложные профессиональные сервисы
- необходимость ручного ввода
- неудобство для пожилых людей

#### Разрыв «История vs Назначение»
Существующие приложения отслеживают срок годности лекарств, но не анализируют эффективность лечения.

### Решение
AI-фармацевтический ассистент, который:

- сканирует лекарства
- анализирует историю пациента
- проверяет совместимость препаратов
- предупреждает о рисках

### Ценность
- снижение медицинских ошибок
- персонализированная медицина
- профилактический подход к лечению

### Монетизация
- freemium-модель
- интеграции с клиниками
- партнёрства со страховыми компаниями


## 3️⃣ SafeCamp — Безопасный отдых

### Проблема
Родители выбирают детские лагеря без достоверной информации:

- накрученные отзывы
- отсутствие базы проверок
- неизвестный уровень безопасности

### Решение
Платформа анализа рисков лагерей.

Функционал:
- сканирование договора или названия лагеря
- анализ юридических данных
- проверка судебных дел
- данные проверок надзорных органов
- рейтинг персонала

### Результат
Формируется отчёт риска:
- юридическая надёжность
- безопасность инфраструктуры
- рейтинг сотрудников


## 4️⃣ Parkly / SpotShare / CityPark

### Проблема
В крупных городах водители тратят до 30 минут на поиск парковки.

Последствия:
- пробки
- стресс
- перерасход топлива
- увеличение выбросов CO₂
- простаивающие частные парковочные места

### Решение
Маркетплейс парковок в реальном времени.

Возможности:
- карта занятости парковок
- аренда частных мест (C2C)
- бронирование парковки

### Монетизация
- комиссия за бронирование
- партнёрства с городами

## 5️⃣ StudentPass

### Проблема
Студенты переплачивают за профессиональный софт и AI-сервисы:

- информация о скидках разбросана
- сложная верификация
- университетские лицензии не используются

### Решение
Единая платформа студенческих привилегий.

Функции:
- каталог подписок
- инструкции по активации
- единый интерфейс AI-инструментов
- уведомления о новых программах
- community-база знаний

### Монетизация
- партнёрские программы
- сотрудничество с университетами
- premium-поддержка оформления доступа

## 6️⃣ САПР VSAT

### Проблема
Отсутствует доступное программное обеспечение для проектирования систем спутниковой связи для студентов и исследователей.

### Решение
САПР с системой поддержки принятия решений:

- автоматический подбор параметров
- моделирование систем связи
- интеллектуальные рекомендации

### Целевая аудитория
- студенты-инженеры
- научные лаборатории
- исследовательские проекты


## 7️⃣ Агрегатор студенческих скидок

### Проблема
Студенты тратят много времени на поиск скидок и акций на разных платформах.

### Решение
Единый сайт-агрегатор:

- купоны
- акции
- подтверждённые студенческие скидки

### Преимущество
Все предложения доступны в одном месте.
