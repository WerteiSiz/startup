# Шаг 1. Сегмент, сценарий и целевая «первая победа»

## 1.1 Beachhead-сегмент

| Параметр | Значение |
|----------|---------|
| Кто | Молодые пользователи 18–30 лет, активно использующие цифровые сервисы |
| Где | Онлайн (веб-приложение), мобильные устройства и ПК |
| Почему именно они | Быстро адаптируются к новым продуктам, дают обратную связь |
| Размер сегмента | ≈ 100–150 тыс. пользователей |

---

## 1.2 Сценарий и First Value Moment

| Параметр | Значение |
|----------|---------|
| Сценарий | Пользователь заходит на сайт и начинает использовать сервис |
| Ценностное предложение | Быстрое решение задачи через цифровой инструмент |
| First Value Moment | Первый полученный результат |
| Time to Value | ≤ 3–5 минут |

# Шаг 2. Критические рыночные гипотезы

## 2.1 Основные гипотезы

| Блок | ID | Гипотеза |
|------|----|---------|
| Сегмент | H1 | Пользователи заинтересованы в продукте |
| Ценность | H2 | Продукт решает реальную проблему |
| Канал | H3 | Социальные сети — эффективный канал |
| Конверсия | H4 | ≥ 30% пользователей регистрируются |
| Удержание | H5 | ≥ 25% возвращаются |
| Монетизация | H6 | Пользователи готовы платить |

---

## 2.2 ТОП-3 рисковых гипотез

| Приоритет | Гипотеза | Почему критична | Как проверим |
|----------|----------|----------------|-------------|
| R1 | H1 | Нет спроса — нет проекта | Тест трафика |
| R2 | H2 | Нет ценности — нет удержания | Анализ поведения |
| R3 | H6 | Нет оплаты — нет бизнеса | Soft-offer |

# Шаг 3. Минимальный бизнес-прототип (МБП)

## 3.1 Выбор формата

| Формат | Плюсы | Минусы | Решение |
|--------|------|--------|--------|
| Лендинг | Быстро | Нет продукта | Дополнительно |
| Прототип | Дешево | Не проверяет реальность | Нет |
| Веб-приложение | Проверяет всё | Требует разработки | ✅ Основной |

---

## 3.2 Требования к МБП

- Регистрация пользователей  
- Основной функционал  
- Повторное использование  
- Soft-offer  
- Система аналитики  

---

## 3.3 Целевые действия

| Действие | Метрика |
|----------|--------|
| Регистрация | CR |
| Использование | Активность |
| Возврат | Retention |
| Оплата | Conversion |
