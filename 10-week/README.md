# Шаг 1. Сегмент, сценарий и целевая «первая победа»

## 1.1 Beachhead-сегмент

| Параметр | Значение |
|----------|---------|
| Кто | Молодые люди 18–30 лет (студенты и молодые специалисты), активно проводящие досуг вне дома |
| Где | Крупные города (онлайн-поиск через смартфон) |
| Почему именно они | Часто ищут куда сходить, чувствительны к цене, активно используют цифровые сервисы |
| Размер сегмента | ≈ 100–200 тыс. активных пользователей |

---

## 1.2 Сценарий и First Value Moment

| Параметр | Значение |
|----------|---------|
| Сценарий | Пользователь планирует досуг (встреча, свидание, отдых) → заходит на сайт → выбирает категорию → находит место со скидкой → сохраняет |
| Ценностное предложение (UVP) | Все скидки на досуг в одном месте — быстро найди куда сходить и сэкономь |
| First Value Moment | Пользователь находит подходящее место и понимает выгоду |
| Time to Value (TTV) | ≤ 2–3 минуты |
